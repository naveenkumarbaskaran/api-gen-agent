# api-gen-agent

> Describe your API in plain English — get a full REST + OpenAPI spec back.

## What it does

Takes a natural language description of your API endpoints, generates FastAPI/Express/Spring Boot boilerplate, OpenAPI 3.0 spec, and example curl commands. Ready to run in minutes.

## Quickstart

```bash
pip install api-gen-agent
```

## Usage

```bash
api-gen-agent create "A REST API for a task manager with users, projects and todos" --framework fastapi
```
